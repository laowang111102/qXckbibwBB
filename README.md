# 停车场管理系统 java496

## 项目概述

停车场管理系统是一个基于前后端分离架构的Web应用程序，主要用于实现停车场的数字化管理。系统采用Java和Spring Boot作为后端开发技术，Vue、Axios和Element UI作为前端开发技术。

## 技术栈

### 前端技术

- Vue
- Axios
- ElementUI

### 后端技术

- Spring Boot
- MyBatis-Plus

### 开发环境

- Java 1.8
- Node.js
- MySQL 8
- Maven

## 功能模块

### 普通用户

- 个人信息管理
- 预定停车位
- 缴费信息管理

### 管理员用户

- 用户信息管理
- 车辆信息管理
- 停泊车辆管理
- 车辆进出管理

## 系统角色

- 普通用户：具有基本操作权限，可管理个人信息、预定停车位和缴费信息。
- 管理员用户：具有更高权限，可以进行用户信息、车辆信息、停泊车辆以及车辆进出管理。

## 运行环境

- 服务器：兼容Java 1.8及以上版本的服务器
- 数据库：MySQL 8
- 客户端：现代浏览器

## 部署步骤

1. 配置Java开发环境
2. 安装MySQL数据库
3. 部署后端服务
4. 部署前端服务
5. 配置服务器和数据库连接

## 目录结构

```
├── backend                  # 后端代码
├── frontend                 # 前端代码
├── documentations           # 项目文档
└── resources                # 项目资源（如数据库文件等）
```
